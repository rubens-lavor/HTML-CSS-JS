Array / Vetor - Parte 3 - Métodos da classe array

Método indexOf informa se o elemento está ou não no vetor, se estiver retorna a posição do elemento, se não houver o item retorna -1

Método sort ordena os elementos do vetor

Método reverse inverte os elementos do vetor

Método join retorna todos os elementos de um vetor separados por vírgula e no formato de string

Método concat faz a união dos itens de dois vetores

length retorna quantos elementos tem no array
